# Higgs Boson Diphoton Cross-Section Measurement using ATLAS Open Data

This repository contains my third-year undergraduate particle physics project completed at the University of Manchester.

The project measures the Higgs boson production cross-section in the \(H \rightarrow \gamma\gamma\) decay channel using 13 TeV ATLAS Open Data. The analysis applies photon selection criteria, models the signal with Gaussian fits, models the background using several functional forms, and extracts the Higgs signal from the diphoton invariant mass distribution.

## Project Objectives

- Analyse proton-proton collision data from the ATLAS experiment.
- Optimise event selection cuts.
- Fit signal and background models.
- Measure the Higgs boson production cross-section.
- Estimate statistical and systematic uncertainties.

## Results

Measured cross-section:

**σ(H → γγ) = 65.9 ± 7.67 (systematic) ± 10.4 (statistical) ± 1.38 (luminosity) fb**

The measured value is consistent with Standard Model expectations within the quoted uncertainties.
